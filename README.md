## 📌 What Is QR Phishing?

QR Phishing (QRLishing) is a cyberattack where
attackers hide malicious URLs inside QR codes.
When victim scans the QR code, they are redirected
to a fake page that steals their credentials.

---

## 🎯 What This Project Does

1. Generates a QR code with attacker server IP
2. Victim scans QR code with their phone
3. Fake WiFi login page opens on victim phone
4. Victim enters their name and password
5. YOU GOT HACKED screen appears with Matrix rain
6. Attacker sees all captured data on laptop terminal

## ⚙️ Setup & Installation

*Step 1 - Clone the repo:*
```bash
git clone https://github.com/mrxrobots/qr-phishing-demo.git
cd qr-phishing-demo


Step 2 - Create virtual environment:
python3 -m venv myenv
source myenv/bin/activate


Step 3 - Install dependencies:
pip install flask qrcode[pil]

Step 4 - Run the server:
python app.py


Step 5 - Generate QR code:
python qr_generate.py


Step 6 - Scan QR with phone on same WiFi
